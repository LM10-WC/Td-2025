Configuration du routeur

Interface FastEthernet0/0

  • IP Address : 192.168.1.1/24
  • Status : Active
  • Commandes : bash interface FastEthernet0/0 ip address 192.168.1.1 255.255.255.0 no shutdown

Interface FastEthernet0/1

  • IP Address : 192.168.2.1/24
  • Status : Active
  • Commandes : bash interface FastEthernet0/1 ip address 192.168.2.1 255.255.255.0 no shutdown
